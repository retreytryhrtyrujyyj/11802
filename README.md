# 校园体育场馆（设施）使用管理网站

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=11802

## 技术介绍
Java语言 + ssm框架/springboot框架（后端）+ Vue前端框架（Web）+ Mysql(数据库)

## 开发工具
IDEA或者MyEclipse + Navicat(mysql数据库可视化工具) + Tomcat

## 项目视频演示
- [演示视频1](https://graduation-images.oss-cn-beijing.aliyuncs.com/videos/330%E5%A5%97springboot%E5%BD%95%E5%83%8F/11802_springboot326%E6%A0%A1%E5%9B%AD%E4%BD%93%E8%82%B2%E5%9C%BA%E9%A6%86%EF%BC%88%E8%AE%BE%E6%96%BD%EF%BC%89%E4%BD%BF%E7%94%A8%E7%AE%A1%E7%90%86%E7%BD%91%E7%AB%99%E5%BD%95%E5%83%8F.mp4)

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
